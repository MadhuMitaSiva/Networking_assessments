10. How DHCP server assigns the IP address to the network?
DHCP assigns IP through DORA process
D-> Discover O->Offer R->Request A-> Acknowledgment

D: When a new device is connected to network it discovers the available DHCP server
O: The Discovered DHCP server responds to the device with the IP address
R: The device accepts the IP address and informs that it had received an IP from the discovered server
A: The discovered server sends acknowledgment for finalization of the address.
